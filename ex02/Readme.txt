기능 (service)
	- 회원 가입 기능		=> regist() 메서드
	1) 이메일, 비밀번호, 이름등을 입력 받아야 함 : RegisterRequest
	2) 받아온 정보를 통해서 기능을 구현 : MemberRegisterService
		- 이미 가입된 이메일이 있다면 가입 거부(예외)
		- 가입된 이메일이 아니라면 회원 정보를 담는 객체에 옮겨 담는다 : Member
		- 저장된 회원정보를 저장소에 전송 : MemberDao

회원 가입 기능
1. 클라이언트로 부터 데이터를 받아옴
2. 적절하게 가공
3. Dao에게 넘겨주어야 한다. (DAO의 메서드를 호출) = DAO에 의존적이다.
4. Dao는 넘겨받은 데이터를 DB에 저장
